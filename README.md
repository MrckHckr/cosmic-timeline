Cosmic Timeline: From the Big Bang to the Birth of the Solar System

A CERN‑style, data‑science‑oriented astrophysics project built with Python and Google Colab.

-) Project Goal

Create a scientifically accurate, reproducible, and extensible timeline of the Universe from the Big Bang to the formation of the Solar System, using:

  - data‑driven modeling

  - logarithmic time analysis

  - professional research practices (GitHub, notebooks, documentation)

-) Scientific Scope

  - Cosmology

  - Big Bang

  - Inflation

  - Particle era

  - Nucleosynthesis

-) Astrophysics

  - Dark Ages

  - Population III stars

  - First galaxies

  - Milky Way formation

  - Molecular clouds

  - Protoplanetary disk

  - Solar System formation
